# AGENTS.md

## Project rules
- Make minimal changes.
- Follow existing patterns before introducing new abstractions.
- Do not rewrite unrelated files.
- Do not change public APIs unless explicitly asked.
- Ask before adding new production dependencies.

## Workflow
- First inspect the relevant files.
- Before editing, propose a short plan.
- After editing, summarize changed files and risks.
- Run the most relevant validation command if available.

## Commands
- Install: npm install
- Dev: npm run dev
- Build: npm run build
- Lint: npm run lint
- Typecheck: npm run typecheck
- Test: npm test
